# Basic Cable Screencap add-on

Headless-Chromium dashboard snapshots for the
[Basic Cable](https://github.com/dbdmdbdmdbdm/basic-cable) app's Home
Dashboard channels. See DOCS for options.

Prefer plain Docker (no HA OS)? The same server ships as a container in
[`companion/ha-screencap`](https://github.com/dbdmdbdmdbdm/basic-cable/tree/main/companion/ha-screencap).

`server.js` here is the canonical copy; `companion/ha-screencap/server.js`
mirrors it — keep them in sync.
